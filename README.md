# Asher

### 커밋 메세지 규칙</br>
Title: `Type: 내용`</br>
Body: 세부 변경사항 설명</br>
Footer: close #이슈번호
| Type | 내용 |
| ------------ | ------------- |
| Feat | 새로운 기능 추가 또는 기능 업데이트 |
| Fix | 버그 또는 에러 수정 |
| Style | 코드 포맷팅, 코드 오타, 함수명 수정 등 스타일 수정 |
| Refactor | 코드 리팩토링(기능 변화 없이 코드만 개선) |
| File | 파일 이동 또는 제거, 파일명 변경 |
| Design | 디자인, 문장 수정 |
| Comment | 주석 수정 및 삭제 |
| Chore | 빌드 수정, 패키지 추가, 환경변수 설정 |
| Docs | 문서 수정, 블로그 포스트 추가 |
| Hotfix | 핫픽스 수정 |
